# 📊 Analysis & Design of Algorithms Lab

## 📌 Overview
This repository contains solutions for ADA Lab assignments. It demonstrates algorithm performance analysis using Python with real-time execution measurements and graphical visualization.

---

## 🚀 Features

### 🔹 Task 1: Algorithm Growth
- O(1), O(n), O(n²), O(log n)
- Execution time comparison
- Graph plotting using matplotlib

### 🔹 Task 2: Searching Algorithms
- Linear Search
- Binary Search
- Best, Average, Worst case analysis

### 🔹 Task 3: Recursion Analysis
- Factorial
- Fibonacci (basic idea included)
- Function call counting

### 🔹 Task 4: Recurrence Relations
- T(n) = T(n/2) + n
- T(n) = 2T(n/2) + n
- Call tracking and execution time

---

- Matplotlib
- Time module
- Random module
